# Northwind Traders SQL Practice Solutions

This repository contains my solutions for the **Northwind Traders** database challenges. Northwind is a classic sample database representing a global import/export company, used to practice complex joins and business reporting.

## 📊 Database Schema
The database consists of several interconnected tables, including:
- **Customers**: Company names and contact information.
- **Orders**: Sales transaction headers (dates, shipping info).
- **Order_Details**: Line items for each order (products, unit price, quantity).
- **Products**: Inventory items, categories, and supplier links.
- **Employees**: Staff details and reporting hierarchy.
